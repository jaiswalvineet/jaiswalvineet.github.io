Rename the file extenstion

May be used to change the file name from mobile phone recording so you can search the file based on mobile no on windows

Power shell script to change naming of this format 0d20170620183415p+91xxxxxxxxx.abc to 91xxxxxxxxx-0d20170620183415p.abc


$orgPath = "C:\Users\[User]\test\"
$delim = "+"
$delimDot = "."

Get-ChildItem $orgPath -Name | `
foreach { 
  Write-Output $_
  $nameArray = $_.Split($delim)
  if($nameArray[1] -gt 0)
  {
      $extArray = $nameArray[1].Split($delimDot)
      $newName = $extArray[0]+'-'+$nameArray[0]+'.'+$extArray[1]
        Write-Output $_
      Rename-Item $orgPath$_ $newName
  }
  Write-Output $newName
}
